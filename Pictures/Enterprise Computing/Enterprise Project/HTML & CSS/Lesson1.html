Quantitative Data

Definition: Numerical data that can be measured, counted, and statistically analyzed. Answers questions like how many, how much, or how often.

  • Examples: Sales, profits, revenue, temperature, web traffic
  • Types: Discrete (whole numbers), Continuous (any value in range)
  • Measurement Tools: Surveys with numeric scales, sensors
  • Data Format: CSV, Excel, SQL
  • Representation: Line, bar, histogram, scatter plots
  • Strengths: Easy comparison, suitable for trends and forecasting

Qualitative Data

Definition: Descriptive, non-numerical data capturing qualities and characteristics. Answers what, why, or how.

  • Examples: Opinions, emotions, interview responses
  • Types:
    • Nominal - unordered categories (e.g., gender)
    • Ordinal - ordered categories (e.g., rankings)
  • Measurement Tools: Open-ended surveys, focus groups
  • Data Format: Strings, transcripts, images
  • Representation: Pie/bar charts, word clouds, tag maps
  • Strengths: Depth and context, reveals human behavior

Key Differences Summary

Feature Quantitative Data Qualitative Data
Nature Numerical Descriptive/Text-based
Purpose Measure, compare, analyze Explain, interpret, explore
Data Format Numbers (integers, decimals) Words, symbols, images
Methods of Analysis Statistical/mathematical Thematic/content analysis
Common Tools Excel, SQL, Tableau NVivo, ATLAS.ti
Examples Sales, age, income Feedback, comments
